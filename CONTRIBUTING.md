## Contributing

Contributions are welcome! To ensure a smooth process, please follow these steps:

1. **Fork the Repository**: Click the 'Fork' button at the top right of this page.
2. **Clone your Fork**: `git clone https://github.com/YOUR_USERNAME/YOUR_PROJECT.git`
3. **Sync with Upstream**: Before creating a branch, make sure your fork is up to date:
   - `git remote add upstream https://github.com/albdangarcia/animal-shelter.git`
   - `git fetch upstream`
   - `git merge upstream/main`
4. **Create a Branch**: `git checkout -b feat/your-feature-name`
5. **Make Changes**: Implement your feature or fix.
6. **Push to your Fork**: `git push origin feat/your-feature-name`
7. **Open a Pull Request**: Go to the original repository and click "Compare & pull request" to submit your changes for review.
